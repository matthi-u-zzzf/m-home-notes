# m-home-notes

## 项目简介

m-home-notes 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面通常以笔记、工具页面、静态演示或小型信息页等形式存在，彼此之间没有强关联，也不归属于某个特定站点或域名。仓库本身仅作为这些页面的集中存储与版本管理地。

## 目录说明

```
m-home-notes/
├── pages/          # 存放所有独立 HTML 页面文件
├── assets/         # 页面引用的公共资源（图片、样式、脚本等）
├── index.html      # 可选：页面索引或导航页
└── README.md       # 本文件
```

- **pages/**：每个子目录或文件对应一个独立页面，命名尽量反映内容主题。
- **assets/**：若页面需要共享资源（如通用 CSS、JS 库、图标等），可放置于此，由各页面按需引用。
- **index.html**：可自行创建简单的索引页，方便浏览所有归档页面（非必需）。

## 页面归档说明

- 每个 HTML 页面均为独立、自包含（或引用 assets 下共享资源）的静态文件。
- 页面内容可能涵盖个人笔记、临时工具、实验性演示、信息整理等，不保证长期维护或更新。
- 归档目的仅为保存与公开，不构成对页面内容准确性、完整性的承诺。
- 部分页面可能包含过时或不准确的信息，请自行判断使用。

## 维护说明

- 本仓库由个人维护，更新频率不固定。
- 欢迎提出 issue 或 pull request 修正页面中的明显错误或失效链接。
- 所有页面均以原始 HTML 形式存储，不涉及后端或动态内容。
- 仓库不提供在线演示或托管服务，页面需通过 GitHub Pages 或其他静态托管方式部署后访问。
